## npm larni o'rnatish

```bash
$ npm install
```

## environment o'zgaruvchilarni o'zgartirish

```bash
# .env.example file nomini .env ga o'zgartiring
$ mv .env.example .env


# .env filedagi kalitlarni o'zingizniki bilan o'zgartiring
$ nano .env

```

## serverni yoqish

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
